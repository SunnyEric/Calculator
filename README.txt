# Calculator
Simple calculate
可以进行
加    +      例 2+3+4
减    -      例 2-3-4
乘    *      例 2*3*4
除    /      例 2/3/4
次方  ^      例 2^3=8
阶层  !      例 3!=6
log   '      例 2'4=log(2,4)=2
ln    ln     例 ln2
sin   sin    例 sin30=0.5
cos   cos    例 cos60=0.5
tan   tan    例 tan45=1
使用括号     例 1+(2-3)*4/5
使用π  pi    例 pi+2
使用e  e     例 lne

使用上一步的结果
例 2+3+4=9
ans-5=4

修改精确度
输入命令numalen，可修改整数长度，范围100~40000
输入命令numblen，可修改小数精度，范围1~100
输入命令quit，即可退出
